# Overages Calculator (Next.js, Node 22)

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy (Vercel)
- Set **Project → Settings → General → Node.js Version: 22.x**
- Default Build Settings:
  - Install: `npm install`
  - Build: `npm run build`
  - Output: `.next`
- If you ever change settings, use **Redeploy → Clear build cache**.
